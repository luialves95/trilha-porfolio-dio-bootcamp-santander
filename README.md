## 📌 Sobre o projeto

Portfólio pessoal desenvolvido para o Bootcamp Santander da DIO, com integração à API pública do GitHub. O site exibe informações do perfil (avatar, seguidores, repositórios) e lista os projetos atualizados automaticamente.

## 🎨 Design

- Fundo escuro com detalhes em roxo (`#9b4dff`) e verde (`#00e676`)
- Totalmente responsivo e acessível (teclado, leitores de tela)
- Layout moderno com cards e gradientes

## 🚀 Tecnologias

- HTML5 semântico
- CSS3 (Flexbox, Grid, variáveis)
- JavaScript (Fetch API, manipulação do DOM)
- GitHub REST API

## 📂 Estrutura
- index.html # Página principal
- style.css # Estilos responsivos
- README.md # Este arquivo
- script.js # Aqui temos uma integração em JavaScript
